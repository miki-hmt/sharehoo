WML <prev> 标签

WML 参考手册

定义和用法

<prev> 任务表示返回前一个访问过的卡片。如果不存在前一个卡片，规定 不会有任何效果。

注释：<prev>..</prev> 标签之间的内容是可选的。如果未规定任何内容，则必须使用 <prev/>。

语法

<prev>
..<setvar>..
</prev>

属性

属性 值 描述
class cdata 为此元素设置的 class 名。这个 class 名对大小写敏感。一个元素可链接多个 class。claas 属性中的多个 class 名通过空格进行分隔。
id id 为此元素设置的唯一名称。

实例

<?xml version="1.0"?>
<!DOCTYPE wml PUBLIC "-//WAPFORUM//DTD WML 1.1//EN"
"http://www.wapforum.org/DTD/wml_1.1.xml">

<wml>
<card> 
<p> 
 <do type="accept" label="Back"> 
  <prev> 
   <setvar name="x" value="30"/> 
  </prev> 
 </do> 
</p> 
</card> 
</wml>

没有内容的 <prev> 标签：

<?xml version="1.0"?>
<!DOCTYPE wml PUBLIC "-//WAPFORUM//DTD WML 1.1//EN"
"http://www.wapforum.org/DTD/wml_1.1.xml">

<wml>
<card>
<do type="accept" label="Back">
 <prev/>
</do> 
<p>
Hello World!
</p>
</card>
</wml>
