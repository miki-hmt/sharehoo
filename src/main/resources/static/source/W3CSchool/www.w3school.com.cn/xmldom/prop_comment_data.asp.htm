XML DOM data 属性

Comment 对象参考手册

定义和用法

data 属性返回注释的文本内容。

语法：

commentNode.data

实例

以下代码段使用 JavaScript 函数 loadXMLDoc() 把 XML 文件 books_comment.xml 载入 xmlDoc 中，并输出第一个 <book> 元素的注释文本内容：

xmlDoc=loadXMLDoc("books_comment.xml");

x=xmlDoc.getElementsByTagName("book")[0].childNodes;

for (i=0;i<x.length;i++)
{ 
if (x[i].nodeType==8)
  { 
  //Process only comment nodes
  document.write(x[i].data);
  document.write("<br />");
  } 
}

以上代码的输出：

(Book 6) (Hardcover)

在本例中，我们使用一段循环和 if 语句来执行只针对 comment 节点的处理。comment 节点的节点类型是 8。
