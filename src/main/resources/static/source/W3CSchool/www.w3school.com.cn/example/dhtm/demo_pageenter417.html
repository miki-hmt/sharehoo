This page contains a "Page-Enter" meta tag

Insert the following code line into the <head> section, and your web page will achieve the same effect:

<meta http-equiv="Page-Enter" content="revealtrans(duration=2,transition=17)">
  • The content attribute describes the effect
  • The duration property describes how many seconds before the effect is completed
  • The transition property describes which actual effect your document will have, from 0-23
