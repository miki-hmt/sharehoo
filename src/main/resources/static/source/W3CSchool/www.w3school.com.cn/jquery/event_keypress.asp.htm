定义和用法

keypress 事件与 keydown 事件类似。当按钮被按下时，会发生该事件。它发生在当前获得焦点的元素上。

不过，与 keydown 事件不同，每插入一个字符，就会发生 keypress 事件。

keypress() 函数触发 keypress 事件。或者如果设置了 function 参数，则规定当发生 keypress 事件时执行的代码。

语法

$(selector).keypress(function)
参数 描述
function

可选。规定当按键时要运行的函数。

如果未设置该参数，则 keypress() 函数触发 keypress 事件。

提示和注释

注释：如果在文档元素上进行设置，则无论元素是否获得焦点，该事件都会发生。
