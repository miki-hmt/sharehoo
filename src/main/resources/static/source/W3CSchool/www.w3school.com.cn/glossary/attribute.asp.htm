W3C 术语 - attribute

W3C 术语表中以字母 A 开头的条目

attribute

中文：属性

解释

属性是 DTD 中声明的元素的参数。DTD 中定义了属性的类型和值的范围，包括可能的默认值。

来源

来自：XHTML 1.0: The Extensible HyperText Markup Language (Second Edition) (2000-01-26)

该来源中的术语

attribute

中文：属性

解释

与元素相关的值，包括名称以及相关的（文本）值。

来源

来自：Glossary of Cascading Style Sheets, level 2 CSS2 Specification (1998-05-12)

该来源中的术语

attribute

中文：属性

解释

本文档在这样的 XML 语境中使用术语 "attribute" ：一个元素可以拥有一系列属性（attribute）规范（引用自 XML 1.0 specification [XML] section 3）。

来源

来自：User Agent Accessibility Guidelines 1.0 (2002-12-17)

该来源中的术语

attribute

中文：属性

解释

用于规定有关 SGML 或 XML 元素类型的属性的参数。它是按照属性名称、属性类型以及一个默认值的方式定义的。可以在开始标签中规定定义元素类型的值。

来源

来自：Mathematical Markup Language (MathML) Version 2.0 (2001-02-21)

该来源中的术语

attribute

中文：属性

解释

本文档使用的属性 "attribute" 与在 SGML 和 XML ([XML]) 中使用的类似：可以通过任何属性的属性定义元素类型。一些属性对于内容的可访问性是必须的。（例如，HTML 中的 "alt"、"title" 以及 "longdesc" 属性。）

来源

来自：Authoring Tool Accessibility Guidelines 1.0 (2000-02-03)

该来源中的术语

attribute

中文：属性

解释

对象的显著特征。对象的属性用于描述对象。经常按照对象的物理特性来规定真实世界的对象的属性，比如尺寸、形状、重量、颜色等。电脑空间中的对象的属性可能通常描述大小、编码类型、网络地址等。[WSIA Glossary]

来源

来自：Web Services Glossary (2004-02-11)

该来源中的术语

attribute

中文：属性

解释

对象的特性。在第六章中，该术语指的是具体的 XML 句法结构；XML 的 name="value" 部分。

来源

来自：Resource Description Framework (RDF) Model and Syntax Specification (1999-02-22)
