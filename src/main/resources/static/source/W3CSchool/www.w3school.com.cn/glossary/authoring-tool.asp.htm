W3C 术语 - authoring tool

W3C 术语表中以字母 A 开头的条目

authoring tool

中文：创作工具

解释

“创作工具”是用于制作供 Web 上发布的内容的任何软件。创作工具包括：为制作 Web 内容而特殊设计的编辑工具（例如所见即所得的 HTML 和 XML 编辑器）；提供以 Web 格式保持材料的选项的工具；将文档转换为 Web 格式的工具（例如将桌面发布格式转换为 HTML 的过滤器）；制作多媒体的工具，特别是在希望于 Web 上使用的地方（例如视频制作和编辑套件，SMIL 创作工具包）；供站点管理或站点发布的工具，包括从数据库动态地自动生成网站的工具，以 on-the-fly 方式转换，以及网站发布工具；布局管理工具（例如 CSS 格式化工具）。

来源

来自：Authoring Tool Accessibility Guidelines 1.0 (2000-02-03)

该来源中的术语

authoring tool

中文：创作工具

解释

HTML 编辑器，文档转换工具，从数据库生成 Web 内容的工具都属于创作工具。参阅 "Authoring Tool Accessibility Guidelines" ([WAI-AUTOOLS]) 获得更多有关开发易用工具的信息。

来源

来自：Web Content Accessibility Guidelines 1.0 (1999-05-05)
