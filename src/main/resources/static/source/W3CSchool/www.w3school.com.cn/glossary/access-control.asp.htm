W3C 术语 - access control

W3C 术语表中以字母 A 开头的条目

access control

中文：访问控制

解释

选择性地控制允许谁获得或操作其中（比如 Web 服务器）的信息的能力。

来源

来自：Glossary of "Weaving the Web" (1999-07-23)

该来源中的术语

access control

中文：访问控制

解释

对未授权访问资源的保护；根据某种安全策略管制资源使用的方法，只有已授权的系统实体按照该安全策略才能得到获准对资源的使用。

[RFC 2828]

来源

来自： Web Services Glossary (2004-02-11)
