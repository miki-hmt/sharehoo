按位左移运算符 (<<)

左移表达式的位。

result = expression1 << expression2

参数

result

任何变量。

expression1

任何表达式。

expression2

任何表达式。

说明

<< 运算符把 expression1 的所有位向左移 expression2 指定的位数。例如： 

var temp
temp = 14 << 2

变量 temp 的值为 56，因为 14 （即二进制的 00001110）向左移两位等于 56 （即二进制的 00111000）。

要求

版本 1
