试图对RegExp类型对象之外的其他对象调用 RegExp.prototype.toString 或 RegExp.prototype.valueOf方法。这种调用的类型必须是 RegExp 对象。

要改正此错误

  • 在 RegExp 对象类型 中，只能调用RegExp.prototype.toString 或 RegExp.prototype.valueOf  方法
