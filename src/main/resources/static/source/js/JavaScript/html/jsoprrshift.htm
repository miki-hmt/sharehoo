按位右移运算符 (>>)

右移表达式的位，保持符号不变。

result = expression1 >> expression2

参数

result

任何变量。

expression1

任何表达式。

expression2

任何表达式。

说明

>> 运算符把 expression1 的所有位向右移 expression2 指定的位数。expression1 的符号位被用来填充右移后左边空出来的位。向右移出的位被丢弃。例如，下面的代码被求值后，temp 的值是 -4：-14 （即二进制的 11110010）右移两位等于 -4 （即二进制的 11111100）。 

var temp
temp = -14 >> 2

要求

版本 1
