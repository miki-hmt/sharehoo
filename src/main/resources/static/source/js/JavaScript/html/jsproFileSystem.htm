FileSystem 属性

返回指定驱动器所使用的文件系统的类型。

object.FileSystem

其中 object 总是一个 Drive 对象。

说明

可能的返回类型包括 FAT、NTFS、和 CDFS。

下面这段代码说明了 FileSystem 属性的用法： 

function ShowFileSystemType(drvPath)
{
   var fso,d, s;
   fso = new ActiveXObject("Scripting.FileSystemObject");
   d = fso.GetDrive(drvPath);
   s = d.FileSystem;
   return(s);
}
