sub 方法

将 HTML 的 <SUB> 标识放置到 String 对象中的文本两端。

strVariable.sub( )
"String Literal".sub( )

说明

下面的示例演示了 sub 方法是如何使用的： 

var strVariable = "This is a string object";
strVariable = strVariable.sub( );

在执行上一条语句后 strVariable 的值为： 

<SUB>This is a string object</SUB>

不检查该标识是否已应用于该字符串了。

要求

版本 1
