abs 方法

返回数字的绝对值。

Math.abs(number)

必选项number 参数是要需要计算绝对值的数值表达式。

说明

返回的值是 number 参数的绝对值。

示例

下面的例子演示了abs 方法的用法。 

function ComparePosNegVal(n)
{
   var s;
   var v1 = Math.abs(n);
   var v2 = Math.abs(-n);
   if (v1 == v2)
      s = n + " 和 "
      s += -n + " 的绝对值是相同的。";
   return(s);
}
