Throw 的后面跟有一个表达式，且在同一源代码行上

使用了 throw 关键字，但在同一行内没有紧接一个表达式。一个 throw 语句由两部分组成：throw 关键字，后面紧跟待抛出的表达式。例如；

if (denominator == 0) {
throw new DivideByZeroException();
}

不能将这两个组件分成两行。

要改正此错误

  • 请确保 throw 关键字和待抛出的表达式在同一行。
