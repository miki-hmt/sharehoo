E:checked

  • 版本：CSS3

语法：

E:checked { sRules }

说明：

匹配用户界面上处于选中状态的元素E。(用于input type为radio与checkbox时)

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
