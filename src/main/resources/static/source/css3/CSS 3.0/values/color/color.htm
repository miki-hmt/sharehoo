<color>

  • 版本：CSS1/CSS3

说明：

<color> 可以是一个关键字或者数值规则。
  • <color> 包括 Color Name(颜色名称), HEX, RGB, RGBA, HSL, HSLA, transparent, currentColor

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
