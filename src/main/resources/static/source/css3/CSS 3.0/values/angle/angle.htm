<angle>

  • 版本：CSS3

说明：

<number>接角度单位。
  • 角度值的正常范围应在[0-360deg]内，例如：-10deg与350deg是等价的
  • 角度单位包括有： deg, grad, rad, turn

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
