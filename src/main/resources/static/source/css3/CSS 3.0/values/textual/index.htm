CSS Textual Data Types

  • CSS文本取值

文本值 CSS Textual Data Types

名称 版本 描述
inherit CSS2.1 该值使得属性能够继承祖先设置。
initial CSS3 属性初始值。
unset CSS3 擦除属性申明。
<string> CSS2 字符串(含转义字符串)。
<url> CSS2 在网页上提供一个资源地址。（图像，声频，视频或浏览器支持的其他任何资源）
<identifier> CSS2 使用用户自定义标识名作为组件取值。

紫色 链接表示该属性在CSS3有修改或者增加了新的属性值；橙色 链接表示该属性是CSS3新增属性
