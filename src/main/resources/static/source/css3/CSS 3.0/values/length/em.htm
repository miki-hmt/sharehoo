em

  • 版本：CSS1

说明：

相对长度单位。相对于当前对象内文本的字体尺寸。
  • 如当前对行内文本的字体尺寸未被人为设置，则相对于浏览器的默认字体尺寸。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
