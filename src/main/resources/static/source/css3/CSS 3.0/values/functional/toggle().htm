toggle()

  • 版本：CSS3

语法：

toggle() = toggle(<value>#)

说明：

允许子孙元素使用取值序列中的值循环替换继承而来的值。

  • 每级ul使用不同的标记：

    示例代码：

    ul { list-style-type: toggle(disk, circle, square, box); } em { font-style: toggle(italic, normal); }

    在上述代码中，定义一个多级的ul，第一级使用disk markers，子孙级依次使用circle, square, box。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
