mm

  • 版本：CSS1

说明：

毫米（Millimeters）。绝对长度单位。
  • 1in = 2.54cm = 25.4 mm = 101.6q = 72pt = 6pc = 96px

示例代码：

div { font-size: 10mm; } p { height: 100mm; }

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
