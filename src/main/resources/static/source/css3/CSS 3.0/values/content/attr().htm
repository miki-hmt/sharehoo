attr()

  • 版本：CSS2/3

语法：

attr() = attr(attr-name)

说明：

插入元素的属性值。
  • 在CSS2.1中attr()总是返回一个字符串。在CSS3中attr()可以返回多种不同的类型。
  • 在CSS2.1中可以这样使用：

    示例代码：

    a:after { content: "("attr(href)")"; font-size: 12px; } <a href="http://css.doyoe.com/">CSS参考手册</a>

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
