border-image-outset

  • 版本：CSS3
  • 媒体：视觉

语法：

border-image-outset：[ <length> | <number> ]{1,4}

默认值：0

适用于：所有元素，除table元素设置了 <' border-collapse '> 为collapse之外

继承性：无

动画性：否

计算值：指定值

取值：

<length>：
用长度值指定宽度。不允许负值。
<number>：
用浮点数指定宽度。不允许负值。

说明：

置或检索对象的边框背景图的扩展。
  • 该属性用于指定边框图像向外扩展所定义的数值，即如果值为10px，则图像在原本的基础上往外延展10px再显示。
  • 对应的脚本特性为borderImageOutset。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
