column-count

  • 版本：CSS3
  • 媒体：视觉

语法：

column-count：<integer> | auto

默认值：auto

适用于：除table外的非替换块级元素, table cells, inline-block元素

继承性：无

动画性：否

计算值：指定值

取值：

<integer>：
用整数值来定义列数。不允许负值
auto：
根据 <' column-width '> 自定分配宽度

说明：

设置或检索对象的列数
  • 对应的脚本特性为columnCount。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
