Media Queries: scan

  • 版本：CSS3
  • 是否接受min/max：否

语法：

scan：progressive | interlace

取值：

progressive：
连续扫描
interlace：
交织扫描

说明：

定义电视类设备的扫描工序。
  • 本特性不接受min和max前缀。

  • 简单列举几个应用示例：

    @media tv and (scan:progressive){ … }
@import url(example.css) tv and (scan:interlace);

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
支持版本\类型 IE Firefox Safari Chrome Opera
较早版本 6-8 4 5.1.7 13-23 11.5
较近版本 9
