unicode-bidi

  • 版本：CSS2
  • 媒体：视觉

语法：

unicode-bidi：normal | embed | bidi-override | isolate | isolate-override | plaintext

默认值：normal

适用于：所有元素

继承性：无

动画性：否

计算值：特定值

取值：

normal：
对象不打开附加的嵌入层，对于内联元素，隐式重排序跨对象边界进行工作。
embed：
对象打开附加的嵌入层，<' direction '> 属性的值指定嵌入层，在对象内部进行隐式重排序。
bidi-override：
严格按照 <' direction '> 属性的值重排序。忽略隐式双向运算规则。

说明：

用于同一个页面里存在从不同方向读进的文本显示。与 <' direction '> 属性一起使用
  • 对应的脚本特性为unicodeBidi。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
