flex-flow

  • 版本：CSS3
  • 媒体：视觉

语法：

flex-flow：<' flex-direction '> || <' flex-wrap '>

默认值：看各分拆属性

适用于：flex容器

继承性：无

动画性：否

计算值：指定值

取值：

<' flex-direction '>：
定义弹性盒子元素的排列方向。
<' flex-wrap '>：
控制flex容器是单行或者多行。

说明：

复合属性。设置或检索弹性盒模型对象的子元素排列方式。
  • 对应的脚本特性为flexFlow。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
