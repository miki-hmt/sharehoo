page-break-after

  • 版本：CSS2
  • 2015.8.12

语法：

page-break-after：auto | always | avoid | left | right

默认值：auto

适用于：所有元素

继承性：无

动画性：否

计算值：指定值

媒体：页面

取值：

auto：
如果需要，在对象之后插入页分割符
always：
始终在对象之后插入页分割符
avoid：
避免在对象之后插入页分割符
left：
在对象之后插入页分割符直到它到达一个空白的左页边
right：
在对象之后插入页分割符直到它到达一个空白的右页边

说明：

检索或设置对象之后出现的页分割符。
  • 对应的脚本特性为pageBreakAfter。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
Values IE Firefox Chrome Safari Opera iOS Safari Android Browser Android Chrome
Basic Support 6.0+ 2.0+ 4.0+ 6.0 15.0+ 6.0+ 2.1+ 18.0+
