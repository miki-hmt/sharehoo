text-indent

  • 版本：CSS1/3
  • 媒体：视觉

语法：

text-indent：[ <length> | <percentage> ] && hanging? && each-line?

默认值：0

适用于：块容器

继承性：有

动画性：当使用值为非关键字时

计算值：指定值

取值：

<length>：
用长度值指定文本的缩进。可以为负值。
<percentage>：
用百分比指定文本的缩进。可以为负值。
each-line：
定义缩进作用在块容器的第一行或者内部的每个强制换行的首行，软换行不受影响。（CSS3）
hanging：
反向所有被缩进作用的行。（CSS3）

说明：

检索或设置对象中的文本的缩进。
  • 内联对象要使用该属性必须先使该对象表现为块级或内联块级。
  • hanging和each-line关键字紧随在缩进数值之后

    示例代码：

    div{text-indent:2em each-line;}

    以上代码将使得div内部的第一行及每个强制换行的首行都拥有2em的缩进

  • 对应的脚本特性为textIndent。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
