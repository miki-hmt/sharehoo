zoom

  • 版本：CSS3
  • 媒体：视觉

语法：

zoom：normal | <number> | <percentage>

默认值：normal

适用于：所有元素

继承性：有

动画性：是

计算值：绝对长度

取值：

normal：
使用对象的实际尺寸。
<number>：
用浮点数来定义缩放比例。不允许负值
<percentage>：
用百分比来定义缩放比例。不允许负值

说明：

设置或检索对象的缩放比例。
  • 对应的脚本特性为zoom。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
