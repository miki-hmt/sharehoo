appearance

  • 继承性：无

语法：

默认值：none

适用于：所有元素

说明：

设置或检索外观按照本地默认样式
  • 如果你想去除元素本身的外观并进行自定义，可以设置为none
  • 对应的脚本特性为appearance。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
