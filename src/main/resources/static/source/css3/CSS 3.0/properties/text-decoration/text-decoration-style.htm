text-decoration-style

  • 版本：CSS3
  • 媒体：视觉

语法：

text-decoration-style：solid | double | dotted | dashed | wavy

默认值：solid

适用于：所有元素

继承性：无

动画性：否

计算值：指定的值

取值：

solid：
实线
double：
双线
dotted：
点状线条
dashed：
虚线
wavy：
波浪线

说明：

检索或设置对象中的文本装饰线条的形状。
  • 对应的脚本特性为textDecorationStyle。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
