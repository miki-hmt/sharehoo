jQuery [attribute] 选择器

jQuery 选择器参考手册

实例

选择带有 id 属性的所有元素：

$("[id]")

亲自试一试

定义和用法

[attribute] 选择每个带有指定属性的元素。

可以选取带有任何属性的元素（对于指定的属性没有限制）。

语法

$("[attribute]")
参数 描述
attribute 必需。规定要查找的属性。
