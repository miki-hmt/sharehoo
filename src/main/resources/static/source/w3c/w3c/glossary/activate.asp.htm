W3C 术语 - activate

W3C 术语表中以字母 A 开头的条目

activate

中文：激活（启用）

解释

在本文档中，动词 "to activate" 具有以下含义：执行或实施一个或多个与已启用的元素相关的行为。执行或实施一个或多个与用户代理的用户界面组件相关的行为。激活的效果取决于用户交互控制的类型。例如，当链接被激活时，用户代理通常会取回被连接的 Web 资源。当表单元素被激活时，它可能会改变状态（比如复选框）或者接受用户输入（比如文本输入字段）。

来源

来自：User Agent Accessibility Guidelines 1.0 (2002-12-17)
