URL 函数库含有供处理 URL 的函数。

WMLScript URL 库的函数

函数 描述
escapeString() 用转义序列替换 URL 中的特殊字符。
getBase()  
getFragment() 返回 URL 中的片段。
getHost() 返回 URL 中规定的主机名。
getParameters() 返回 URL 的最后一个路径片段中的参数。
getPath() 返回 URL 中规定的路径。
getPort() 返回 URL 中规定的端口号。
getQuery() 返回 URL 中的查询部分。
getReferer()  
getScheme() 返回 URL 中的 scheme。
isValid() 如果 URL 的语法正确，则返回 true，否则返回 false。
loadString() 返回指定 URL 的内容和内容类型。
resolve() 根据一个基准 URL 和 一个相对 URL 返回一个绝对 URL。
unescapeString() 用字符替换 URL 中的转义序列。
