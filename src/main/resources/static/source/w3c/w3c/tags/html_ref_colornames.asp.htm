HTML 颜色名

  • HTML 文档类型
  • HTML 字符集

本页提供了被大多数浏览器支持的颜色名。

提示：W3C 的 HTML 4.0 标准仅支持 16 种颜色名，它们是：aqua、black、blue、fuchsia、gray、green、lime、maroon、navy、olive、purple、red、silver、teal、white、yellow。

如果使用其它颜色的话，就应该使用十六进制的颜色值。

颜色名列表
