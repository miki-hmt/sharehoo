HTML 5 <ol> 标签

实例

两个不同的有序列表：

<ol>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

<ol start="50">
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ol>

亲自试一试

浏览器支持

所有浏览器都支持 <ol> 标签。

定义和用法

<ol> 标签定义有序列表。有序列表可以是数字或字母顺序。

请使用 <li> 标签来定义列表项。

提示和注释

提示：对于无序列表，请使用 <ul> 标签。

提示：请使用 CSS 来设置列表的样式。

HTML 4.01 与 HTML5 之间的差异

在 HTML 4.01 中不赞成使用 "start" 和 "type" 属性，但是在 HTML5 中支持。

"reversed" 属性是 HTML5 中的新属性。

在 HTML 4.01 中不赞成使用 "compact" 属性，在 HTML5 中不支持。

属性

New：HTML5 中的新属性。

属性 值 描述
compact compact

HTML5 中不支持。HTML 4.01 中不赞成使用。

规定列表呈现的效果比正常情况更小巧。
reversed reversed 规定列表顺序为降序。(9,8,7...)
start number 规定有序列表的起始值。
type
  • 1
  • A
  • a
  • I
  • i
 规定在列表中使用的标记类型。

全局属性

<ol> 标签支持 HTML 5 中的全局属性。

事件属性

<ol> 标签支持 HTML 5 中的事件属性。
