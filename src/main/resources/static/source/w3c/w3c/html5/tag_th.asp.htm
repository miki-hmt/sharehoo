HTML 5 <th> 标签

定义和用法

<th> 标签定义 HTML 表格中的表头单元格。

HTML 表格有两种单元格类型：

  • 表头单元格 - 包含头部信息（由 th 元素创建）
  • 标准单元格 - 包含数据（由 td 元素创建）

th 元素中的文本呈现为粗体并且居中。

td 元素中的文本是普通的左对齐文本。

实例

一个简单的 HTML 表格，带有两个表头单元格和两个数据单元格：

<table border="1">
  <tr>
    <th>月份</th>
    <th>存款</th>
  </tr>
  <tr>
    <td>一月</td>
    <td>3000 元</td>
  </tr>
</table>
