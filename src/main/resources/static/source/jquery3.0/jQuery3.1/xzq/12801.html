element

概述    element selector

返回值：Array<Element(s)>

描述：根据给定（html）标记名称选择所有的元素。

  • V : 1.0jQuery( "element" )

    element: 一个用来搜索的元素。 指DOM节点的标签名。

调用JavaScript的 getElementsByTagName()函数，当该表达式使用时返回相应的元素。

示例

实例

查找一个 DIV 元素。

HTML 代码: 
<div>DIV1</div>
<div>DIV2</div>
<span>SPAN</span>
jQuery 代码: 
$("div");
结果:
[ <div>DIV1</div>, <div>DIV2</div> ]
