[attribute|=value]

概述    attributeContainsPrefix selector

返回值：Array<Element(s)>

描述： 选择指定属性值等于给定字符串或以该字符串为前缀（该字符串后跟一个连字符“-” ）的元素。

  • V : 1.0jQuery( "[attribute|='value']" )

    attribute: 一个属性名.

    value: 一个属性值，引号是可选的. 可以是一个 有效标识符或带一个引号的字符串。

这个选择器是引入CSS规范来处理语言属性。

示例

实例

查找hreflang属性值是英语的所有链接。 

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>attributeContainsPrefix demo</title>
  <style>
  a {
    display: inline-block;
  }
  </style>
  <script src="https://code.jquery.com/jquery-1.10.2.js"></script>
</head>
<body>
 
<a href="example.html" hreflang="en">Some text</a>
<a href="example.html" hreflang="en-UK">Some other text</a>
<a href="example.html" hreflang="english">will not be outlined</a>
 
<script>
$( "a[hreflang|='en']" ).css( "border", "3px dotted green" );
</script>
 
</body>
</html>
