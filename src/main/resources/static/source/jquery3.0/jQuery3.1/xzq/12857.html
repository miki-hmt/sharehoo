:disabled

概述    disabled selector

返回值：Array<Element(s)>

描述：选择所有被禁用的元素。

  • V : 1.0jQuery( ":disabled" )

如同其他伪类选择器（那些以“:”开始），建议前面加上一个标记名称或其他选择器;否则，通用（通配）选择器("*")被默认使用。换句话说，$(':disabled')等价于$('*:disabled')，所以应该使用$('input:disabled')或类似写法代替。

虽然他们的选择的结果通常是一样的，:disabled选择器和[disabled] 属性（attribute） 选择器还是有些细微的差别； :disabled 检查元素实际的disabled属性值（手册网注：是否被禁用，布尔值true/false），[disabled] 检查是否存在的disabled属性。

:disabled 选择器仅限用于选择支持disabled属性（attribute）的HTML元素(<button>, <input>, <optgroup>, <option>, <select>, <textarea>, <menuitem>, and <fieldset>)。

示例

实例

查找所有不可用的input元素

HTML 代码: 
<form>
  <input name="email" disabled="disabled" />
  <input name="id" />
</form>
jQuery 代码: 
$("input:disabled")
结果: 
[ <input name="email" disabled="disabled" /> ]
