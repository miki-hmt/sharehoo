ancestor descendant

概述    descendant selector

返回值：Array<Element(s)>

描述：选择给定的祖先元素的所有后代元素。

  • V : 1.0jQuery( "ancestor descendant" )

    ancestor: 任何有效的选择器。

    descendant: 一个用来筛选后代元素的选择器。

一个元素的后代可能是该元素的一个孩子，孙子，曾孙等。

示例

实例

找到表单中所有的 input 元素

HTML 代码: 
<form>
  <label>Name:</label>
  <input name="name" />
  <fieldset>
      <label>Newsletter:</label>
      <input name="newsletter" />
 </fieldset>
</form>
<input name="none" />
jQuery 代码: 
$("form input")
结果: 
[ <input name="name" />, <input name="newsletter" /> ]
