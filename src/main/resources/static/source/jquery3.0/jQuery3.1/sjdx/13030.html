event.currentTarget

概述    event.currentTarget

返回值：Element

描述： 在事件冒泡过程中的当前DOM元素。

  • V : 1.3event.currentTarget

这个属性总是等于函数的this。

如果您使用jQuery.proxy或另一种形式操作作用域，this 将等于您所提供的context（上下文），而不是event.currentTarget

示例

警报currentTarget的匹配`this`关键字。 

$("p").click(function(event) {
  alert( event.currentTarget === this ); // true
});
