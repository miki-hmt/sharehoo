jQuery.isWindow()

概述    jQuery.isWindow( obj )

返回值：Boolean

描述：确定参数是否为一个window对象。

  • V : 1.4.3jQuery.isWindow( obj )

    • obj
      类型: PlainObject
      用于测试是否为一个window对象

这个方法在jQuery很多地方用到，用来确定我们的操作是否为一个浏览器窗口操作。（如当前窗口或一个iframe）。

示例

判断传入的参数是否为 window。 

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-latest.js"></script>
</head>
<body>
  Is 'window' a window? <b></b>
<script>$("b").append( "" + $.isWindow(window) );</script>
 
</body>
</html>
