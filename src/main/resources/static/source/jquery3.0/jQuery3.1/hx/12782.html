jQuery(callback)

概述    jQuery( callback )

返回值：jQuery

描述：当DOM完成加载的时候绑定一个要执行的函数。

  • V : 1.0jQuery( callback )

    • callback
      类型: Function()
      当DOM ready的时候执行的函数。

$(document).ready()的简写。

这个函数的作用如同$(document).ready()一样，只不过用这个函数时，需要把页面中所有需要在 DOM ready时执行的 $()操作符都包装到其中来。从技术上来说，这个函数是可链式调用的－－但真正以这种方式链接的情况并不多。

示例

实例

当DOM加载完成后，执行其中的函数。

jQuery 代码: 
$(function(){
  // 文档就绪
});

实例

使用 $(document).ready() 的简写，同时内部的 jQuery 代码依然使用 $ 作为别名，而不管全局的 $ 为何。

jQuery 代码: 
jQuery(function($) {
  // 你可以在这里继续使用$作为别名...
});
